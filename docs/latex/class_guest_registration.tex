\doxysubsection{Detailed Description}
The \doxylink{class_guest_registration}{Guest\+Registration} class represents a registration for a guest attendee. 

This class inherits from the \doxylink{class_registration}{Registration} class and adds additional functionality specific to guest registrations. It provides methods to calculate the registration fee and convert the registration details to a string format. \begin{DoxySeeAlso}{See also}
