\doxysubsection{Detailed Description}
The \doxylink{class_main_window}{Main\+Window} class represents the main application window. 

This class inherits from QMain\+Window and provides the user interface for the application. It contains various widgets such as a table view, line edit, push button, and actions. The \doxylink{class_main_window}{Main\+Window} class also manages the registration model, registration list, and proxy model.
