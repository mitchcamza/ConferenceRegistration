\doxysubsection{Detailed Description}
Manages a list of conference registrations. 

The \doxylink{class_registration_list}{Registration\+List} class provides functionality to add and query conference registrations. It ensures no duplicate email addresses exist, unless they are associated with different names. It also calculates total fees based on registration types, as well as the number of attendees registered from a specific institution/affiliation.
