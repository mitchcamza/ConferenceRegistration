\doxysubsection{Detailed Description}
The \doxylink{class_registration_model}{Registration\+Model} class is a custom model that extends QStandard\+Item\+Model. 

This class provides additional functionality for managing \doxylink{class_registration}{Registration} objects in a model. It allows adding, removing, and sorting registrations, as well as updating the data in the model.
